Pac-Man S2D Project

University Project | C++ | S2D

About this project

This project was to create a game similar to Pac-Man using C++ and S2D. The project started with me rendering a Pac-Man and some simple movement within the game's level. Over time I developed simple AI for the game, implemented sound effects, added lives and a simple hiscores document that could be read and written to.
The project has areas for development such as including multiple levels, collectables, and powerups.

Developed

This project was my first project using C++ and helped me understand the differences between C++ and JavaScript which I have a background with from my work.

Review

Overall I'm happy with the current development of the project, there are still a large number of changes that I'd like to implement to the project during my personal time to help further develop the project and develop my skills.
