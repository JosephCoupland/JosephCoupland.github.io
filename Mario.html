Mario SDL Project

University Project | C++ | SDL | Marked 90/100

About this project

This project is to create a game similar to Mario using C++ and OpenGL. The project started with me rendering a placeholder background image which would hold the scene within the game level. Over time I rendered characters to the screen, gave them controls and also created a tile map that the characters could interact with. The project was developed to include enemies and audio as well as a playable Luigi.
The project has areas for development such as including multiple levels, collectables, powerups, saving and loading.

Developed

This project has helped me understand the use of having additional libraries such as SDL and made me appreciate classes and inheritance, and has also improved my debugging

Review
